# KCC: Korean Civil Case Dataset

KCC is a benchmark dataset for legal information retrieval based on publicly accessible Korean civil court decisions. It provides expert-annotated graded relevance labels for evaluating case retrieval and similarity modeling in statutory-law systems.

## Intended Use
KCC is intended for research and evaluation in legal information retrieval. It is not designed as a balanced training dataset.
